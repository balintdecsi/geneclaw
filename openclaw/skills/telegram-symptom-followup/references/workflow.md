# Workflow details

## Onboarding

Ask for:

- genetic-test website
- username
- password

Wait until all are provided, then start Round 1 immediately.

## Message style

Keep questions short and yes/no.

Prefer direct summaries like:

- “Got it — for this round I have…”
- “I still need yes/no for…”
- “Recorded.”

## Before Round 5

Do **not**:

- give a disease overview
- say which disease is leading
- conclude early unless emergency escalation is required

## Suggested follow-up question patterns

### GI / colorectal pattern

Use 1–2 of:

- ongoing constipation/diarrhea or bowel-pattern change
- narrow stools
- feeling like stool is still there after going
- severe abdominal swelling or blockage signs

### Cardiac pattern

Use 1–2 of:

- pain spreading to arm/jaw/neck/shoulder
- worse even at rest
- shortness of breath
- dizziness/fainting

### Rheumatology pattern

Use 1–2 of:

- prolonged morning stiffness
- swollen/tender/warm joints
- reduced hand function / grip / mobility

## Decision threshold

A pattern is strong enough when high-importance or specific symptoms persist across multiple rounds and competing patterns are weaker.

## DocFinder output

Return:

- short statement that this is not a diagnosis
- likely pattern
- German specialist type
- 3–8 doctors with city/district and profile links

## Vienna examples

For bowel/colorectal patterns in Vienna, suitable DocFinder specialty terms include:

- Gastroenterologe
- Gastroenterologin
- Innere Medizin und Gastroenterologie
